## Interaction States

### Button - Hover
```
Streamlit Default:
- Color slightly shifts
- Cursor changes to pointer
- Subtle shadow increase
```

### Service Cards - Hover (CSS)
```
Transform: translateY(-5px)
Box-shadow: increased
Opacity: slight increase
Transition: 0.3s cubic-bezier(0.4, 0, 0.2, 1)
```

### Back Button - Click
```
Navigation to home screen
State reset
Page refresh (st.rerun())
```

### CTA Button - Click
```
Landing state set to False
Booking form appears
Page refresh (st.rerun())
```

## Responsive Breakpoints

### Desktop (1200px+)
- Hero: Full width, 100px padding
- Services: 3 columns
- Trust: 3 columns side-by-side

### Tablet (768px-1200px)
- Hero: Full width, 60px padding
- Services: 2 columns
- Trust: 2 columns + 1 below

### Mobile (< 768px)
- Hero: Full width, 40px padding
- Services: 1 column (stacked)
- Trust: 1 column (stacked)
- Font sizes: Reduced 10-15%
- Spacing: Reduced proportionally

## Animation Timeline

```
Page Load:
  1. Hero section fades in (0.3s)
  2. Trust cards cascade in (0.2s each, staggered)
  3. Service cards appear with slight delay (0.5s)
  4. CTA section slides up (0.4s)
  5. Footer badges fade in (0.3s)

CTA Click:
  1. Button appears pressed (0.1s)
  2. Landing page fades out (0.3s)
  3. Booking form fades in (0.3s)

Back Button Click:
  1. Page fades out (0.2s)
  2. Home screen loads (0.3s)
```
